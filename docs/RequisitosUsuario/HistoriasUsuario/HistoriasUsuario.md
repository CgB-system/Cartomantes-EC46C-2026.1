
# 1. História de Usuário

A Tabela 3 a seguir contém as Histórias de Usuário elicitadas.

| ID   | História de Usuário | Critérios de Aceitação | Prioridade | RF/RNF Relacionado | Story Points |
|------|---------------------|------------------------|------------|--------------------|--------------|
| US01 | Como fã de TCG, eu quero escolher uma base pronta de carta para não precisar configurar a estrutura visual do zero. | 1. O sistema deve exibir ao menos 3 tipos de bases (RPG, TCG e Especial).<br>2. Ao clicar em uma base, os campos de edição devem ser carregados automaticamente. | Alta | - | 5 |
| US02 | Como mestre de RPG, eu quero editar os atributos numéricos da carta para definir o poder e a resistência da minha criatura. | 1. Deve haver campos específicos para Nome, Ataque e Defesa.<br>2. O sistema deve validar se os campos de atributos contêm apenas números. | Alta | - | 5 |
| US03 | Como criador de cartas, eu quero escolher uma imagem do meu dispositivo para personalizar a arte central da carta. | 1. Deve permitir upload de arquivos .png e .jpg.<br>2. A imagem deve ser redimensionada automaticamente para caber na dimensão da carta. | Média | - | 3 |
| US04 | Como colecionador, eu quero selecionar as dimensões da carta para que ela caiba nos meus protetores (sleeves). | 1. O sistema deve apresentar um menu com dimensões em milímetros.<br>2. A visualização da carta deve se ajustar à proporção escolhida. | Média | - | 3 |
| US05 | Como usuário frequente, eu quero salvar minhas cartas em uma conta pessoal para continuar editando-as futuramente. | 1. Deve haver um botão "Salvar" visível.<br>2. As cartas devem aparecer em uma galeria chamada "Galeria". | Alta | - | 8 |
| US06 | Como jogador, eu quero exportar minha carta em PDF para poder imprimi-la com alta qualidade. | 1. O PDF deve ter resolução mínima de 300 DPI.<br>2. O download deve iniciar após clicar em "Exportar". | Alta | - | 5 |
| US07 | Como criador de baralhos, eu quero duplicar uma carta existente para criar variações rapidamente. | 1. Deve existir opção "Duplicar".<br>2. A cópia deve herdar todos os dados da original. | Média | - | 3 |
| US08 | Como membro da comunidade geek, eu quero compartilhar o link da minha carta para mostrar para amigos. | 1. O sistema deve gerar um link público único.<br>2. Deve haver botões de compartilhamento. | Média | - | 2 |
